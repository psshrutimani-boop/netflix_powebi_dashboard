# netflix_powebi_dashboard
Netflix Movies Analysis Dashboard (Power BI)

Project Overview

This project presents an interactive dashboard built using Microsoft Power BI to analyze Netflix movie data.
The dashboard provides insights into movie release trends, distribution by era, and key statistics about the dataset.

Dashboard Features

- Total Movies KPI – Displays the total number of movies in the dataset.
- Latest Movie Year KPI – Shows the most recent movie year available.
- Movies Released Trend – A line chart showing how movie releases changed over time.
- Movies Distribution by Era – A pie chart displaying movies grouped by era.
- Year Filter (Slicer) – Allows users to filter the dashboard by movie year.
- Movie Details Table – Displays movie names and their release years.

Tools Used

- Microsoft Power BI
- Data Visualization
- DAX (Data Analysis Expressions)

Dataset

The dataset contains information about Netflix movies including:

- Movie ID
- Movie Name
- Release Year
- Era Category

Files in this Repository

- "netflix_movies_dashboard.pbix" – Power BI dashboard file
- "netflix_dataset.csv" – Dataset used for analysis
- "dashboard_screenshot.png" – Screenshot of the dashboard
- "dashboard_demo.mp4" – Screen recording of dashboard interaction (optional)

Key Insights

- Movie releases have increased significantly in recent years.
- Most movies in the dataset were released after 2000.
- The dashboard allows quick exploration of trends using filters.

How to Use

1. Download the ".pbix" file.
2. Open it in Microsoft Power BI Desktop.
3. Interact with the slicers and charts to explore the data.
